# Balangay

## Project Overview
Balangay is a full-stack web application we developed as students for the "Pangarap sa Bayan" hackathon to provide a digital barter platform where small businesses can exchange services, surplus inventory, or storage, and informal workers can freely trade their skills. Recognizing that unexpected crises often leave people with no cash on hand, we designed this system to tackle real-world financial bottlenecks by creating a dedicated space for mutual aid and market visibility. By combining a responsive Next.js and React front-end with a reliable Express and Firebase backend, we wanted to ensure that entrepreneurs and individuals always have a secure, accessible way to sustain their operations and livelihoods through community-driven trade.

## Technologies & Frameworks Used
This application was built utilizing a modern JavaScript stack to ensure scalability and rapid development:
- **Next.js**: The core React framework used for server-side rendering, routing, and overall application architecture.
- **React**: Utilized for building interactive and reusable user interface components.
- **Express.js**: Handled our custom backend REST API routes and server logic.
- **Firebase**: Implemented for fast, secure database management (Firestore) and seamless user authentication.
- **Tailwind CSS**: Used for utility-first styling to create a clean, responsive, and modern user interface without writing custom CSS files.
